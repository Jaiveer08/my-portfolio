What the Desktop Software Does

The Python desktop client runs on each lab machine to continuously capture the active window title, top network-consuming process, and upload/download speeds using psutil and pygetwindow. Data is pushed to Firebase Realtime Database in real time. The client runs background threads to ensure low overhead and uses a minimal Tkinter UI for configuration.

What the Web App Does

The Flask-based admin dashboard authenticates staff via Firebase Authentication, reads live system data from Firebase, and displays a lab overview with per-system activity and bandwidth. Administrators can remotely trigger controls (e.g., put all systems to sleep) and monitor recent activity to manage labs effectively.
